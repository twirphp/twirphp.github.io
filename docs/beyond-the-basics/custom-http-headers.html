Read HTTP Headers from responsesâ€‹

Twirp client responses are structs that depend only on the protobuf response. HTTP headers can not be used by the Twirp client in any way.

However, remember that the TwirPHP client is instantiated with a PSR-18 HTTP client. You can implement a custom client that captures information coming from headers.
